This is how you use the python library:

Install the AmselPy python library to take control:

# import library
from amselpy import Amsel

# create an instance
amsel = Amsel()

# set adress
amsel.use("")

# control movements
amsel.forward(120)
amsel.sleep(5)
amsel.stop()

Checkout the documentation on https://baumeise.github.io/amsel/docs for further information.
