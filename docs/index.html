# About

This is the official Python library for the Amsel robot.

# Usage

Get started with the amselpy library. 

pip install amselpy

# import library
from amselpy import Amsel

# create an instance
amsel = Amsel()

# set adress
amsel.use("<YOUR-AMSELS-IP>")

# control movements
amsel.forward(120)
amsel.sleep(5)
amsel.stop()
